Project 01

ERP Implementation — Odoo ERP

ERP Odoo Centralized Data
1 / 2

ออกแบบและ implement ระบบ ERP ระดับองค์กรเพื่อเชื่อม process ข้ามหลายแผนก ภายใต้รูปแบบ centralized ลดงาน manual และเพิ่ม data visibility ให้ทั้งองค์กร

Objective

สร้างรากฐาน ERP ที่ใช้งานได้จริงและสอดคล้องกับการทำงานประจำวันของ Sales, Procurement, HR, Engineering และ Production Planning

Key Actions

  • วิเคราะห์และ redesign business process workflow
  • ออกแบบ ERP system architecture ให้สอดคล้องกับการทำงานภายใน
  • เชื่อมหลายแผนกผ่าน centralized database
  • ลดขั้นตอนการทำงานแบบ manual
  • พัฒนา custom module เพื่อรองรับ organization-specific workflow

Business Impact

ทำให้องค์กรมี single source of truth สำหรับข้อมูล ลดงานซ้ำซ้อนระหว่างแผนก และเพิ่มความสามารถด้าน data-driven decision making

90,000 THB / year ประหยัดค่าใช้จ่ายระบบ HR
60,000 THB / year ประหยัดค่าใช้จ่ายระบบ Sales
Project 02

Handheld Scanner Production System

Shopfloor Digitization Real-time Data

พัฒนา handheld workflow สำหรับ shopfloor เพื่อแทนที่การจดข้อมูลบนกระดาษ ด้วยการบันทึกเข้าระบบโดยตรง ทำให้การทำงานเร็วขึ้นและเพิ่ม real-time visibility

Objective

Redesign วิธีเก็บข้อมูลการผลิต ให้ operator บันทึกงานได้ตรงจุดปฏิบัติงาน แทนการกลับไปกรอกข้อมูลแบบ delayed manual entry ที่คอมพิวเตอร์แยก

Key Actions

  • วิเคราะห์ flow การทำงานเดิมที่ใช้กระดาษ
  • ออกแบบ process ใหม่แบบ scan-based สำหรับ setup, running, pause และ downtime
  • เชื่อม handheld workflow เข้ากับ database structure และ production dashboard
  • พัฒนา mobile-friendly UI สำหรับการใช้งานจริงบน shopfloor

Business Impact

ลด human error ลดขั้นตอนการกรอกข้อมูลแบบ manual เพิ่ม real-time production visibility และทำให้ workflow ต่อเนื่องขึ้น

Project 03

Production Monitoring System

Industrial IoT Dashboard Analytics
1 / 2

พัฒนา industrial IoT monitoring solution เพื่อแสดงข้อมูลการผลิตแบบ real-time โดยแปลง machine-level signal ให้เป็น structured record และ actionable dashboard

Objective

เพิ่ม production visibility ด้วยการเก็บ sensor-based event และแปลงเป็น operational data ที่ฝ่ายบริหารใช้วิเคราะห์และวางแผนได้

Key Actions

  • ออกแบบ signal collection flow จาก sensor ไปยัง database
  • สร้าง structured activity logging สำหรับ production event
  • พัฒนา data output สำหรับ dashboard และ analytics
  • สนับสนุน hourly production visibility และ productivity tracking

Business Impact

ติดตาม machine productivity ได้ชัดขึ้น วิเคราะห์ hourly performance ได้ดีขึ้น และเพิ่มความแม่นยำในการวางแผนการผลิตผ่าน real-time visibility

Project 04

IT Cost Optimization Project

Cost Reduction Vendor Negotiation Infrastructure

ปรับโครงสร้าง printer infrastructure ด้วยแนวคิด business-case ทั้งการเปรียบเทียบต้นทุน การสื่อสารกับผู้บริหาร และการ redesign solution ร่วมกับ vendor

Objective

ลด IT operating cost และเพิ่มประสิทธิภาพในการบริหารอุปกรณ์ โดยเปลี่ยนจากการใช้งาน printer แบบกระจัดกระจายไปสู่รูปแบบที่ควบคุมได้และคุ้มค่ากว่า

Key Actions

  • วิเคราะห์ต้นทุนของ printer environment เดิม
  • จัดทำ comparison model และ calculation logic
  • นำเสนอ recommendation ให้ทีมผู้บริหาร
  • เจรจากับ vendor และเสนอ solution ที่มีประสิทธิภาพมากกว่าเดิม

Business Impact

สร้างโอกาสลดต้นทุนประมาณ 300,000 THB ต่อปี พร้อมทำให้ printer infrastructure เรียบง่ายขึ้นและบริหารแบบ centralized ได้ดีขึ้น

300,000 THB / year ประหยัดจาก printer migration และ infrastructure optimization

Project 05

Network Infrastructure Redesign

Security VLAN Architecture
1 / 2

Redesign enterprise network architecture เพื่อแยก critical infrastructure, production system, surveillance และ guest access ออกจากกัน พร้อมเพิ่ม security control และ operational reliability

Objective

ยกระดับ security posture ด้วย segmentation ที่ชัดขึ้น inter-zone access control ที่ดีขึ้น และแยก business-critical กับ public-access environment

Key Actions

  • ออกแบบ VLAN-based network segmentation
  • กำหนด office, server, meeting room, CCTV, OT, guest และ backup zones
  • ปรับปรุง firewall-centered architecture และ routing control
  • ลดความเสี่ยงจาก cross-network exposure ที่ไม่จำเป็น

Business Impact

เพิ่มความปลอดภัยของ IT infrastructure ลดความเสี่ยงจาก lateral movement เพิ่มความเสถียรของ network และทำให้ architecture ชัดเจนขึ้นสำหรับการขยายในอนาคต
